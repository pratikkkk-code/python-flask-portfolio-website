# Portfolio Website with Flask

## Overview

A personal portfolio website built using Flask, HTML, and CSS. The website displays personal information, skills, projects, and includes a contact form.

## Features

- Flask Routing
- HTML Templates
- CSS Styling
- Contact Form
- Dynamic Message Display

## Technologies Used

- Python
- Flask
- HTML
- CSS

## Project Structure

portfolio-website/
│
├── app.py
├── templates/
│   └── index.html
├── static/
│   └── style.css
└── README.md

## Run

pip install flask

python app.py

Open:

http://127.0.0.1:5000
